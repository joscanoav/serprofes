# 🚀 Bootcamp Exercises Repository

Este repositorio contiene los ejercicios, prácticas y ejemplos desarrollados durante mi proceso de aprendizaje en el **Bootcamp de Desarrollo Web**.

El objetivo principal de este repositorio es documentar mi progreso, reforzar los conceptos aprendidos en clase y disponer de una colección organizada de ejercicios que sirvan como referencia en el futuro.

---

## 📚 Contenido del repositorio

Actualmente, el repositorio incluye ejercicios relacionados con las siguientes tecnologías:

### 🌐 HTML
- Estructura básica de páginas web.
- Uso de etiquetas semánticas.
- Formularios.
- Tablas y listas.
- Buenas prácticas de maquetación.

### 🎨 CSS
- Selectores y propiedades.
- Modelo de caja (Box Model).
- Flexbox.
- Grid Layout.
- Diseño responsive.
- Estilos para formularios y componentes.

### ⚡ JavaScript
- Variables y tipos de datos.
- Operadores.
- Condicionales.
- Bucles.
- Funciones.
- Arrays y objetos.
- Manipulación del DOM.
- Eventos.
- Ejercicios prácticos de lógica.

### 🟢 Node.js
*(En desarrollo)*

Actualmente estoy incorporando ejercicios relacionados con Node.js, incluyendo:

- Introducción al entorno de ejecución.
- Uso de módulos.
- Gestión de paquetes con npm.
- Creación de scripts.
- Lectura y escritura de archivos.
- Desarrollo de pequeños programas desde la terminal.

## ▶️ Cómo utilizar este repositorio

1. Clona el repositorio:

```bash
git clone https://github.com/tu-usuario/tu-repositorio.git
```

2. Accede al proyecto:

```bash
cd tu-repositorio
```

3. Explora las carpetas correspondientes a cada tecnología y ejecuta los ejercicios según sea necesario.

Por ejemplo, para ejecutar un ejercicio de Node.js:

```bash
node nombre-del-archivo.js
```
